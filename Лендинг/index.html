ПО инженерного анализа
Цели и задачи
Цель проекта
Целью нашего проекта является разработка подпроектов, объединенных темой прочностного анализа.
Задачи
  1. Создание концепции проекта;
  2. Поиск информации о прочностном расчете;
  3. Изучение литературы, сайтов, статей;
  4. Выбор ПО для реализации работ;
  5. Проведение расчетов;
  6. Написание методических указаний;
  7. Создание отчетности по проекту.
Работы
Перевод с Fortran на C# кода реализации решения дифференциального уравнения второго порядка
Перевод с Fortran на C# кода реализации решения дифференциального уравнения второго порядка, взятого из книги Жермен-Лакур П., Жорж П. Л., Пистр Ф., Безье П. "Математика и САПР". Добавлено подробное описание математической модели, а также комментарии по переводу с одного языка программирования на другой.
Перевод с Fortran на C# кода реализации решения уравнения методом Гаусса-Зейделя
Перевод с Fortran на C# кода реализации решения уравнения методом Гаусса-Зейделя, взятого из книги Жермен-Лакур П., Жорж П. Л., Пистр Ф., Безье П. "Математика и САПР". Добавлено подробное описание математической модели, а также комментарии по переводу с одного языка программирования на другой.
Перевод с Fortran на C# кода реализации решения уравнения методом Рунге-Кутта 4 порядка
Перевод с Fortran на C# кода реализации решения уравнения методом Рунге-Кутта 4 порядка, взятого из книги Жермен-Лакур П., Жорж П. Л., Пистр Ф., Безье П. "Математика и САПР". Добавлено подробное описание математической модели, а также комментарии по переводу с одного языка программирования на другой.
Определение форм и расчет собственных колебаний балок
Проведены расчеты форм собственных колебаний в разработанном ПО. Расчет форм собственных колебаний для тех же балок в Autodesk Fusion 360. Сравнение результатов расчета.
Создание уроков по работе с OpenFOAM
Созданы 4 урока по работе с OpenFOAM для выполнения расчетов гидрогазодинами (ламинарное, турбулентное течение, многофазный поток, конвекция).
Расчёт скруббера Вентури для программного модуля "Скрубберы для Autodesk Inventor 2022"
Создание 3D-модели для параметризации. Проведение программного расчёт и расчёта в Mathcad.
Расчет вала для графической подсистемы для SolidWorks
Сделан прочностной расчет вала при воздействии крутящих моментов и осевых сил программно и в Mathcad. Построены эпюры напряжений и произведено сравнение с аналогичным расчетом в генераторе валов Inventor.
Расчет прямобочных и эвольвентных шлицевых соединений для графической подсистемы для SolidWorks
Сделан проверочный расчет прямобочных и эвольвентных шлицевых соединений
Расчет стальных витых цилиндрических пружин сжатия
ПО "Расчет пружин сжатия", методика расчета стальных витых цилиндрических пружин сжатия и обзор аналогов ПО по расчету пружин
